# 项目说明

## 项目简介

**HTML 精简工具**（原 PwDemo）是一款基于 WPF（.NET 8）的桌面小工具。
它的用途非常聚焦：**把一段动态渲染后、体积庞大的 HTML 代码，精简成只保留
自动化定位所需要的语义信息的"干净 HTML"**，方便人工阅读、AI 解析或作为
后续自动化脚本的输入。

- 运行平台：Windows 10 / 11
- 技术栈：.NET 8、WPF、HtmlAgilityPack
- 交互方式：完全离线，**无需登录、无需浏览器、无需网络**

## 本次具体做了哪些事情

1. **去掉"登录/打开浏览器"流程**
   - 移除了 `Launch Browser` / `Navigate` / `Close Browser` 等按钮。
   - 不再启动 Playwright Chromium 实例，也不再访问任何网址。
   - 改为"**用户手动把动态 HTML 粘贴到输入框**"这种纯离线工作方式。

2. **界面全面中文化**
   - 窗口标题改为 `HTML 精简工具`。
   - 按钮文字改为：`精简 HTML`、`复制结果`、`清空`。
   - 左右两栏标题改为：`原始 HTML（请粘贴到此处）`、`精简后的 HTML`。
   - 状态栏提示文字全部改为中文，如 `就绪` / `正在精简 HTML...` /
     `精简完成（N 字符）` / `已复制到剪贴板` / `已清空` / `请先粘贴 HTML 代码` 等。

3. **重写 HTML 精简逻辑（纯 C#，无需浏览器）**
   - 新建 `Services/HtmlSimplifierService.cs`，使用 `HtmlAgilityPack` 解析 HTML。
   - 删除整段节点：`script`、`style`、`noscript`、`svg`、`link`、`meta`。
   - 删除所有 HTML 注释。
   - 清理属性：
     - 删除所有 `on*` 事件处理属性（如 `onclick`、`onmouseover`）。
     - 删除 `style`、`class` 属性。
     - 只保留白名单属性：`id`、`name`、`type`、`role`、`href`、`src`、
       `alt`、`title`、`value`、`placeholder`、`aria-label`、
       `aria-labelledby`、`aria-describedby`、`data-testid`、`data-test`、
       `data-qa`、`data-cy`、`for`、`action`、`method`，以及所有 `data-*` 属性。
   - 合并文本节点中的连续空白，减小输出体积。

4. **精简 UI 布局**
   - 从原来的"URL 栏 + 四宫格（原始 HTML / 精简 HTML / 元素 JSON / 日志）"
     改为极简的"左右两栏"：左边粘贴输入、右边显示结果。
   - 新增"**复制结果**"按钮，一键把精简后的 HTML 复制到剪贴板。
   - 新增"**清空**"按钮，快速清空输入/输出。

5. **依赖与项目清理**
   - `PwDemo.csproj` 中移除 `Microsoft.Playwright 1.47.0` 依赖。
   - 新增 `HtmlAgilityPack 1.11.67` 依赖。
   - 删除 `Services/PlaywrightService.cs` 和 `Scripts/JsScripts.cs`
     （及其所在目录），代码行数显著下降。
   - 删除了"提取元素 JSON"相关功能（用户目前用不到）。

6. **新增本中文说明文档**
   - 即你现在看到的 `项目说明.md`，说明项目用途、价格以及本次改动明细。

## 使用步骤

1. 用 Visual Studio 或 `dotnet build` 构建并运行项目。
2. 把浏览器中"检查元素"或"查看网页源代码"得到的动态 HTML 全选复制。
3. 粘贴到软件左侧的「原始 HTML」文本框。
4. 点击「**精简 HTML**」按钮，右侧立即出现精简后的结果。
5. 点击「**复制结果**」即可把精简后的 HTML 复制到剪贴板，用于后续处理。
